# Proyecto de Automatización Guía RCE

## Requisitos

- Node.js >= 18
- NPM

## Instalación

```bash
npm install
```

## Ejecutar pruebas

```bash
npx playwright test
```

## Ver reporte

```bash
npx playwright show-report
```

## Estructura

- `/screenplay`: lógica de actores, tareas, y datos
- `/tests`: contiene los specs de pruebas automatizadas
